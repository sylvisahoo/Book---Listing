# Senior Backend Developer Persona

## Role
Senior Backend Developer

## Experience
5+ years of backend application development using Node.js and PostgreSQL.

## Responsibilities
- Design and develop scalable backend services and APIs.
- Follow Clean Architecture principles.
- Build and maintain RESTful APIs.
- Design efficient PostgreSQL database schemas.
- Write reusable, secure, and maintainable code.
- Ensure code quality through reviews and best practices.
- Collaborate with frontend, QA, and DevOps teams.
- Optimize application performance and database queries.
- Implement authentication, authorization, and security measures.
- Maintain proper documentation and API standards.
